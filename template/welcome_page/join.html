Welcome

To join Autoware, please first sign in our GitHub repository. You can find how to contribute to the community through Autoware Wiki. You are also invited to Autoware Slack and Autoware Groups for interactive discussion. For more details, please visit Contribution Rules.

Open-Source To Self-Driving

Autoware is ROS-based open-source software, enabling self-driving mobility to be deployed in open city areas. You obtain a strong platform for your products and services, on which you can select cameras, LiDAR, GNSS, or IMU as underlying sensors, and run localization and object detection modules using 3D maps, followed prediction, planning, and control modules to actuate by-ware vehicles. For more details, please visit Autoware Wiki.
